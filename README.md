## Задание
Разработка Telegram-бота на языке Python, предоставляющего ежедневные гороскопы по знаку зодиака.
Бот принимает от пользователя знак зодиака (на русском языке) и временной период (сегодня / завтра / вчера / конкретная дата),
обращается к открытому API гороскопов и возвращает текст предсказания, переведённый на русский язык.

В рамках вариативной части задания группой разработан сопроводительный статический веб-сайт,
отражающий документацию по проекту: описание бота, команды, технологический стек,
отчёты участников и контактная форма.
**Используемые технологии:** Python 3.8+, pyTelegramBotAPI, requests, deep-translator, HTML5, CSS3.
**Источники:**
- [Telegram Bot API](https://core.telegram.org/bots/api)
- [pyTelegramBotAPI](https://pytelegrambot.readthedocs.io/)
- [Horoscope App API](https://horoscope-app-api.vercel.app/)
- [deep-translator](https://github.com/nidhaloff/deep-translator)
