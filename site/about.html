О проекте

Telegram Horoscope Bot — учебный Telegram-бот на языке Python, разработанный в рамках проектной (учебной) практики по направлению Информационные системы и технологии. Бот предоставляет ежедневные гороскопы для всех двенадцати знаков зодиака с автоматическим переводом на русский язык.

Python 3.8+ pyTelegramBotAPI requests deep-translator Horoscope API v1

Команды бота

После запуска бот регистрирует следующие команды и реагирует на текстовые сообщения:

Как работает /horoscope

Реализовано через цепочку шагов с помощью register_next_step_handler:

1
Пользователь вводит /horoscope — бот запрашивает знак зодиака.
2
Пользователь вводит знак на русском (например: Весы). Бот проверяет его наличие в словаре и запрашивает день.
3
Пользователь вводит СЕГОДНЯ, ЗАВТРА, ВЧЕРА или дату в формате ГГГГ-ММ-ДД.
4
Бот конвертирует русский знак в английский (словарь RU_TO_EN) и русский день в английский (RU_DAY_TO_EN), затем делает GET-запрос к API.
5
Полученный английский текст переводится на русский язык через GoogleTranslator (библиотека deep-translator).
6
Пользователь получает гороскоп на русском языке вместе со знаком зодиака и датой.

Внешнее API гороскопов

  • Endpoint: GET https://horoscope-app-api.vercel.app/api/v1/get-horoscope/daily
  • Параметр sign: название знака на английском (Aries, Taurus, …)
  • Параметр day: TODAY, TOMORROW, YESTERDAY или дата
  • Ответ: JSON с полем data.horoscope_data — текст предсказания на английском
  • Перевод: текст автоматически переводится на русский через Google Translate

Пример запроса

GET /api/v1/get-horoscope/daily?sign=Libra&day=TODAY

Структура проекта

  • site/ — статический сайт (HTML + CSS)
  • subproject/bot.py — основной файл бота
  • subproject/requirements.txt — зависимости Python
  • reports/ — отчёты участников (.docx) и шаблон
  • docs/README.md — техническая документация проекта

Установка и запуск

  1. Перейти в папку subproject/
  2. Установить зависимости: pip install -r requirements.txt
  3. Открыть bot.py и вставить токен от @BotFather в переменную BOT_TOKEN
  4. Запустить: python bot.py
