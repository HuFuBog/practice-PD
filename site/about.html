О проекте

Telegram Horoscope Bot — учебный Telegram-бот на языке Python, разработанный в рамках проектной (учебной) практики по направлению Информационные системы и технологии. Бот предоставляет ежедневные гороскопы для всех двенадцати знаков зодиака, используя открытое внешнее API.

Python 3.8+ pyTelegramBotAPI Horoscope API v1

Команды бота

После запуска бот регистрирует следующие команды и реагирует на обычные текстовые сообщения:

КомандаОписание
/startПриветственное сообщение и показ клавиатуры команд
/helloАналог /start — ещё одно приветствие
/horoscopeЗапуск диалога: знак зодиака → день → гороскоп
/helpСправка по всем доступным командам
/menuПовторный показ кнопочной клавиатуры
Любой текстБот повторяет введённое сообщение (эхо)

Как работает /horoscope

Реализовано через цепочку шагов с помощью register_next_step_handler:

1
Пользователь вводит /horoscope — бот запрашивает знак зодиака.
2
Пользователь вводит знак на русском (например: Весы) — бот запрашивает день.
3
Пользователь вводит СЕГОДНЯ, ЗАВТРА, ВЧЕРА или дату ГГГГ-ММ-ДД.
4
Бот конвертирует русское название знака в английское (словарь RU_TO_EN) и делает GET-запрос к API.
5
Пользователю приходит текст гороскопа, знак и дата в формате Markdown.

Внешнее API гороскопов

  • Endpoint: GET https://horoscope-app-api.vercel.app/api/v1/get-horoscope/daily
  • Параметр sign: название знака на английском (Aries, Taurus, …)
  • Параметр day: TODAY, TOMORROW, YESTERDAY или дата
  • Ответ: JSON с полем data.horoscope_data — текст предсказания

Пример запроса

GET /api/v1/get-horoscope/daily?sign=Libra&day=TODAY

Установка и запуск

  1. Перейти в папку subproject/telegram_bot/
  2. Открыть bot.py и вставить токен от @BotFather в переменную BOT_TOKEN
  3. Запустить: python bot.py
