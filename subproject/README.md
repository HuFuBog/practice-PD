# Telegram Horoscope Bot

## Описание
Telegram-бот на Python, который предоставляет гороскопы по знаку зодиака.
Пользователь выбирает знак зодиака и тип гороскопа:
- ежедневный
- еженедельный
- ежемесячный
После этого бот получает гороскоп через API и переводит его на русский язык.
Поддерживаемые команды:
| Команда | Описание |
|---|---|
| `/start`, `/hello` | Приветствие и показ клавиатуры |
| `/horoscope` | Запуск диалога: знак → тип гороскопа → результат |
| `/help` | Справка по командам |
| `/menu` | Повторный показ клавиатуры |
| Любой неизвестный текст | Сообщение о доступных командах |

## Требования
- Python 3.8 или выше
- Токен Telegram-бота от [@BotFather](https://t.me/BotFather)
- Интернет-соединение (для запросов к API гороскопов и сервису перевода)

## Установка и запуск
1. Перейдите в папку `subproject/`:
   ```bash
   cd subproject
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Откройте файл `bot.py` и вставьте свой токен в переменную `BOT_TOKEN`:
   ```python
   BOT_TOKEN = 'ваш_токен_здесь'
   ```
4. Запустите бота:
   ```bash
   python bot.py
   ```
Бот начнёт принимать сообщения. Откройте чат с ботом в Telegram и введите `/start`.

## Зависимости (`requirements.txt`)
| Библиотека | Назначение |
|---|---|
| `pyTelegramBotAPI` | Работа с Telegram Bot API |
| `requests` | HTTP-запросы к API гороскопов |
| `deep-translator` | Перевод текста гороскопа с английского на русский |

## Как работает /horoscope
1. Пользователь вводит `/horoscope`
2. Бот спрашивает знак зодиака (на русском: Лев, Весы, Рак...)
3. Бот предлагает выбрать тип гороскопа:
   - сегодняшний
   - недельный
   - месячный
4. Бот конвертирует русский знак в английский (`RU_TO_EN`)
5. Выполняется запрос к API гороскопов
6. Полученный текст переводится на русский через `GoogleTranslator`
7. Пользователь получает готовый гороскоп
