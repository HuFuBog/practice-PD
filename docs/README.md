# Документация

- README.md — основной файл с документацией, описывающий процесс выполнения практики.
- При необходимости могут добавляться дополнительные файлы Markdown.

## Как запустить сайт
1. Откройте файл `site/index.html` в любом современном браузере.
2. Навигация реализована как фиксированная панель с эффектом «стекла», ссылки работают без серверных компонентов.
3. При необходимости отредактировать стили – изменяйте `site/css/style.css`.

## Как запустить Telegram‑бот
1. Перейдите в `subproject/`.
2. Установите зависимости (`pip install pyTelegramBotAPI requests`).
3. Добавьте ваш токен в переменную окружения `TELEGRAM_BOT_TOKEN` и запустите `python bot.py`.
