## Описание проекта

**Telegram Horoscope Bot** — Telegram-бот на Python, предоставляющий ежедневные гороскопы
для всех двенадцати знаков зодиака. Пользователь вводит знак на русском языке,
выбирает день (сегодня / завтра / вчера / дата), и бот возвращает текст гороскопа,
автоматически переведённый на русский язык.

## Технологический стек

| Технология | Назначение |
|---|---|
| Python 3.8+ | Язык разработки бота |
| pyTelegramBotAPI | Взаимодействие с Telegram Bot API |
| requests | HTTP-запросы к API гороскопов |
| deep-translator | Перевод текста гороскопа с английского на русский |
| HTML5 / CSS3 | Статический сайт-документация |

## Как запустить сайт

1. Откройте файл `site/index.html` в любом современном браузере.
2. Навигация реализована как фиксированная панель — ссылки работают без серверных компонентов.
3. Для редактирования стилей изменяйте `site/css/style.css`.

## Как запустить Telegram-бот

1. Перейдите в папку `subproject/`.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
   Файл `requirements.txt` содержит: `pyTelegramBotAPI`, `requests`, `deep-translator`.
3. Откройте `bot.py` и вставьте свой токен в переменную `BOT_TOKEN`.  
   Токен выдаётся ботом [@BotFather](https://t.me/BotFather) при создании бота.
4. Запустите:
   ```bash
   python bot.py
   ```

## Команды бота

| Команда | Описание |
|---|---|
| `/start`, `/hello` | Приветственное сообщение и клавиатура команд |
| `/horoscope` | Запуск диалога: знак → день → гороскоп на русском |
| `/help` | Справка по командам |
| `/menu` | Повторный показ клавиатуры |
| Любой текст | Бот повторяет сообщение (эхо) |
